# IntelliCards — AI-Powered Flashcard Generator

IntelliCards is a fully responsive web application that converts educational documents (PDFs, Word documents, text, images) into intelligent, study-ready flashcards and dynamic multi-mode quizzes. Powered by a hybrid text-parsing pipeline, a 4-level segmentation fallback engine, and GPT-4o, the app provides a premium futuristic glassmorphism interface along with study dashboards, streak trackers, and ReportLab PDF/JSON export support.

---

## Key Features

1. **Smart 4-Level Document Segmentation Pipeline**:
   - **Level 1 (Font Size Hierarchy)**: Automatically detects larger headings to segment the text by subtopics.
   - **Level 2 (Bold Indicators)**: Falls back to splitting sections by bolded text or key indicators.
   - **Level 3 (Page-by-Page Splitting)**: Segments text according to page numbers for multi-page documents.
   - **Level 4 (Paragraph Chunking)**: Local paragraph-based grouping when no formatting triggers are present.
2. **AI Flashcard Generation**: Generates structured questions, answers, hints, and code/use-case examples using GPT-4o JSON mode, with a local regex NLP fallback if the API key is not configured.
3. **Adaptive Timed Quizzes**:
   - **Multiple Choice Questions (MCQ)**: 4 options with distractor answers sampled from other cards.
   - **True / False (T/F)**: Statements generated by modifying concepts.
   - **Fill in the Blank**: Word scaffolding to test recall.
4. **Learning Analytics Dashboard**: Tracks overall study accuracy, total generated flashcards, and study streaks, alongside a list of weak topics and recent quiz activities.
5. **Universal Exports**: Download any generated deck as a standard `JSON` document or a beautifully formatted ReportLab `PDF` stylesheet.
6. **Futuristic Glassmorphic UI**: Styled using custom design tokens, harmonious gradients, blur filters, and micro-interactions.

## Installation & Setup

1. **Clone the Repository**:
   ```bash
   git clone <your-repository-url>
   cd IntelliCards
   ```

2. **Set Up Python Virtual Environment**:
   ```bash
   python -m venv venv
   # Activate on Windows:
   .\venv\Scripts\activate
   # Activate on macOS/Linux:
   source venv/bin/activate
   ```

3. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Environment Variables Configuration**:
   Create a `.env` file in the root directory:
   ```env
   OPENAI_API_KEY=your-actual-api-key-here
   SECRET_KEY=any-random-string-for-session-management
   ```
   *Note: If no API key is specified, the application uses a smart local NLP fallback to generate high-quality cards.*

5. **Start Flask Server**:
   ```bash
   python app.py
   ```
   Access the web app at `http://127.0.0.1:5000`.
